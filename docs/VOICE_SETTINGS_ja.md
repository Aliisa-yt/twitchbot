# [CAST] 設定項目の説明

[CAST] セクションで指定する音声設定の書き方をまとめています。[twitchbot.ini](twitchbot.ini) 本体の概要は [CONFIGURATION_ja.md](CONFIGURATION_ja.md) を参照してください。

## 書式

音声設定の書式は次のとおりです。`DEFAULT` の例を示します。

```
DEFAULT = [
    {"lang" : "ja", "engine" : "cevio_ai", "cast" : "さとうささら", "param" : "v45,s50"},
    {"lang" : "all", "engine" : "gtts", "param" : "v95"},
    ]
```

`[]` の中に `{}` で囲んだ設定を複数並べます。末尾の `,` は付けてください。

この例では、チャットメッセージの言語が `"ja"` のときはエンジン `"cevio_ai"`、キャスト `"さとうささら"`、音量 `45`、速度 `50` で読み上げます。それ以外の言語ではエンジン `"gtts"`、音量 `95` を使います。

## 値の説明

1. `"lang"` の値  
    ISO 639-1 言語コードを指定します。チャットメッセージの言語と一致したときにその行が適用されます。全言語に適用したい場合は `"all"` を指定します。

2. `"engine"` の値  
    使用する TTS エンジン名を指定します。利用可能な値:
    - `"gtts"`: gTTS(Google Text-to-Speech) を使用
    - `"bouyomichan"`: 棒読みちゃんを使用
    - `"cevio_cs7"`: CeVIO CS7 を使用
    - `"cevio_ai"`: CeVIO AI を使用
    - `"voicevox"`: VOICEVOX を使用
    - `"coeiroink"`: COEIROINK ver1 系列を使用
    - `"coeiroink2"`: COEIROINK ver2 系列を使用

3. `"cast"` の値  
    エンジンが複数の音声を持つ場合に、利用する音声名を指定します。指定方法はエンジンごとに異なります。`"gtts"` は音声指定を無視するため、省略可能です。

4. `"param"` の値  
    デフォルトと異なるパラメータを指定する場合に使用します。

    ```
    書式: "v(数値),s(数値),t(数値),i(数値),a(数値)"
    ```

    数値の意味はエンジンごとに異なります。対応していない項目は無視されます。順序は任意で、同じ項目を複数書いた場合は最後の値が有効です。

    - v: 音量（例: `v100`）
    - s: 読み上げ速度（例: `s100`）。`"gtts"` は非対応です。
    - t: トーン（例: `t50`）。`"gtts"` は非対応です。
    - i: 声質（例: `i55`）。`"cevio_cs7"`, `"cevio_ai"` のみ対応しています。
    - a: イントネーション（例: `a1.0`）。`"gtts"`, `"bouyomichan"` は非対応です。
